木鱼湖垂钓

2019年12月5日

木鱼湖是我任湖南工程学院党委书记期间，由湖南工程学院出土地，湘潭市人民政府出资金，校地共建的开放性公园。公园如今绿树成荫，水波荡漾，是市民休闲的好地方。偶尔得闲，也去木鱼湖公园甩上几竿。
共建谋筹犹在目，
而今退位扮渔翁。
柳丝拂面遮冬阳，
野鸭追虾入苇丛。
大学城中求学问，
木鱼湖里起禅风。
行人偶遇呼书记，
惊起涟漪碎碧穹。

仰天湖野钓

2019年10月22日

仰天湖位于长株潭接壤的株易路口，中建集团投资150亿元，正在建设绿色养生示范城。今天跑到仰天湖野钓。
养生基地养生湖，
绿柳青莲见野凫。
三五老翁抛诱饵，
无鱼有水也康娱。

钓鱼迷路

2019年11月22日

两次到株洲友谊水库钓鱼，虽有导航，却两次迷路。
觅水寻鱼山径断，
导航引我转圈圈。
人生处处多歧路，
却怪玲优吐嗲言。

注：高德导航的播音是台湾明星林志玲。`

游钓湘潭

2018年9月

到湘潭的水库钓鱼，是一场历史巡游。青年、东风、超美、向红、跃进、向东、钢铁、红旗、勤俭、红日、卫星，这些水库名留下的是深深的时代烙印。
库区巡钓似青年，
苦守东风偃月残。
超美肥鲈玩路亚，
向红湘鲫戏台竿。
鲢鳙跃进吞香饵，
黄尾向东跳浅滩。
前辈胼胝钢铁梦，
红旗依旧漫湘潭。

雨中游盘龙大观园

2017年10月
游园天助兴， 细雨点涟漪。
彩彩鸳鸯羽， 红红锦鲤鳍。
云烟宝塔渺， 草木卦宫漓。
仙鹤歪头瞅， 伊人恐湿衣。

游万楼

2017年9月

大学同窗好友何兄歌劲在电视台多次介绍万楼历史。得闲游万楼，见重修后的崭新万楼，更为惊叹的是，300多年前的杰灵台还苔藓依旧。
台谓杰灵绿藓深，
续承文脉万楼新。
湘江滚滚先贤去，
唯有何兄道古今。
